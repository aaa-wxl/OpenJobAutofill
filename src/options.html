OpenJobAutofill

维护本机简历资料

数据只保存在本机浏览器里，不会走同步云端。AI 只辅助识别表单字段和可匹配的资料项，不会读取你的简历值。点击开始填写后会自动填写能确定的字段，待处理字段可在资料面板查看和手动复制。

首次使用

三步开始网申填写

  1. 填写资料 先在下面按模块维护本机简历资料。
  2. 保存资料 点击底部“保存资料”，看到浮层提示后就已保存到本机。
  3. 开始填写 打开招聘表单页，点击插件图标里的“开始填写”。

AI 页面分析是可选增强：如果希望模型辅助理解表单字段，再到下方 API 设置填写 API Key、Base URL 和模型名。测试 API 连接只验证当前表单值，点击“保存 API 设置”后才会用于填写。不配置 API 时仍可使用本地规则填写。

资料目录

简历资料

资料加载后，编辑任意字段会在这里提示是否需要保存。

版本更新

插件会定期检查 GitHub Release。发现新版本时，浏览器图标会显示 NEW；更新前建议先导出资料备份。更新时不要卸载扩展，覆盖或重新加载后，本机资料和 API 设置会保留。
